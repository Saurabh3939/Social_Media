A fully functional, responsive social media platform built using the **MERN Stack**. This project allows users to create accounts, share their thoughts with images, and interact with other users' posts.

[Image of MERN Stack architecture]

## üöÄ Key Features

- **Secure Authentication**: User Signup and Login functionality using **JWT (JSON Web Tokens)** and **bcrypt** for password hashing.
- **Image Uploads**: Integrated with **Cloudinary** for high-quality, cloud-based image storage and retrieval.
- **Post Management (CRUD)**:
  - Users can create posts with text and images.
  - Real-time feed display of all posts.
  - **Owner-Only Delete**: Users can only delete their own posts, ensuring data security.
- **Interactions**: Like and Comment functionality on posts.
- **Responsive UI**: Modern, card-based layout designed with custom CSS for a professional look and feel.
